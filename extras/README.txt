Welcome to this bundle of extras for the Wrangling F1 Data With R book.

This package contains several files to support the book:

- ergastR-core.R contains R functions for accessing the online ergast API
- ergastdb13.sqlite is a SQLite version of the ergast database to the end of the 2013 season
- scraperwiki.sqlite contains scraped results data from the fomula1.com website and is provided for personal research purposes only

A virtual machine containing a version of RStudio that you can access through your browser and preloaded with the above files is also available. For more details, see: https://github.com/psychemedia/wranglingf1data-docker

For examples of how to use RStudio to explore F1 data, see: https://leanpub.com/wranglingf1datawithr/
